# Enabling GitHub Pages in ORCHUB Repository

## Steps to Enable GitHub Pages

1. **Go to the repository settings**  
   Navigate to the `Settings` tab of the ORCHUB repository.

2. **Find the GitHub Pages section**  
   Scroll down to the `GitHub Pages` section in the settings.

3. **Select Source**  
   In the `Source` drop-down menu, select the branch you want to use for GitHub Pages. Usually, this is the `main` branch.

4. **Choose the folder**  
   You can choose between the root directory or a `/docs` folder if you want to serve your site from there.

5. **Save**  
   Click `Save` to apply the changes.

6. **Access your page**  
   After enabling GitHub Pages, your site will be available at `https://<your-username>.github.io/ORCHUB/`.

## Additional Information
- It may take a few minutes for the changes to take effect.
- You can customize your site further using an `index.html` file and other resources.
